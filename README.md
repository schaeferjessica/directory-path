# Directory Path

Directory Path is a solution to help beginner coders to get the path on their projects. 

## Installation

Install packages with: 

```bash
yarn
```
```bash
npm i
```
Start the app in development mode with:


```bash
yarn dev
```
```bash
npm run dev
```
